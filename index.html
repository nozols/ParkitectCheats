Cheating in parkitect

This mod adds cheats to Parkitect, the ParkitectNexus Client is required to run this Mod.

Features

  • Spawn as many guests as you wish
  • Give yourself money
  • Change the speed
  • Rate the park
  • Toggle rain
  • Set the lightlevel

Controls

  • Windows: T
  • OSX: Option + T
